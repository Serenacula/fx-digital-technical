# What does each bar chart row look like?

## Answer

Each row: hex code label (e.g. `#FF3A20`) followed by a bar. The bar is filled with that colour and its width represents the percentage of total (non-transparent) pixels. Rows sorted most → least dominant.

## Rationale

User's stated design.
